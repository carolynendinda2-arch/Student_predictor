
# 🎓 AI Student Performance Predictor

This is a simple machine learning web application built using Streamlit. It predicts whether a student will PASS or FAIL based on study hours and sleep hours.

---

## 🚀 Live Demo
Add your Streamlit link here after deployment:

---

## 📊 Features
- Predicts PASS or FAIL results
- Uses Machine Learning (Random Forest Classifier)
- Simple web interface using Streamlit
- Real-time predictions based on user input

---

## 🧠 How it works
The model is trained using sample data with:
- Hours studied
- Sleep hours
- Pass/Fail labels

The model learns patterns and makes predictions based on user input.

- Python
- Streamlit
- Pandas
- Scikit-learn
